# Human Development Index (HDI) Predictor

## Project Overview

The Human Development Index (HDI) Predictor is a Machine Learning and Flask-based web application that predicts the Human Development Index (HDI) score of a country using important development indicators.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis, model training, and deployment through a Flask web application.

---

## Features

- HDI Score Prediction
- Human Development Category Classification
- Interactive Flask Web Interface
- Exploratory Data Analysis (EDA)
- Linear Regression Model
- Pickle Model Serialization

## Installation

```bash
pip install -r requirements.txt
```

Run the training script

```bash
python src/train_model.py
```

Run the application

```bash
python src/app.py
```

Open

```
http://127.0.0.1:5000
```

## Output

The application predicts:

- HDI Score
- Human Development Category
  - Very High
  - High
  - Medium
  - Low
